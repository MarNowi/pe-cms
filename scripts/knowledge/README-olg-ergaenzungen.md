# OLG-Ergänzungen

Diese Ergänzungen schließen die durch den Ratgeber `wer-darf-photovoltaikanlagen-installieren` erkannten Wissenslücken.

Enthalten sind:

- Dachdeckerhandwerk
- Dokumentation
- Elektrotechnikerhandwerk
- Handwerksordnung
- Handwerksrolle
- Installateurverzeichnis
- Potentialausgleich

Zusätzlich wird `Solaranlage` als Alias von `Photovoltaik (PV)` aufgelöst.

## Test

```bash
node scripts/knowledge/analyze-olg-ratgeber-complete.mjs
```

Die Ergänzungen verändern weder den bestehenden Ratgeber noch das Frontend-Energielexikon. Sie erweitern zunächst nur PEAK.Knowledge.
