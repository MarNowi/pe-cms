# Kopie des PEAK.Energy-Energielexikons

Die Einträge wurden zunächst unverändert aus dem Repository `MarNowi/pe` übernommen.

## Zweck

- sofortige Nutzung in PEAK.Knowledge
- keine Kopplung zwischen den Repositories
- keine Änderung der bestehenden Website
- spätere Bereinigung und Synchronisierung bleiben möglich

Die vier Chunk-Dateien dienen nur dazu, die große Datenmenge übersichtlich im Repository abzulegen. Der Adapter fasst sie wieder zu einer gemeinsamen Wissensbasis zusammen.
